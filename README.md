# Student Depression Prediction

## Overview
This project uses Machine Learning techniques to analyze and predict student depression based on different factors.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Machine Learning Prediction
- Feature Importance Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Files
- app.py → Main application
- model.pkl → Trained ML model
- scaler.pkl → Data scaler
- test.py → Testing file

## Output
The model predicts whether a student is likely to face depression based on input features.
